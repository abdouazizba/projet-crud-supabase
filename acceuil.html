Présentation du Projet

Technologies utilisées

  • Frontend: HTML, CSS (Tailwind), JavaScript
  • Backend: Supabase (Backend as a Service)
  • Base de données: PostgreSQL avec Row Level Security
  • Authentification: Système d'authentification Supabase

Fonctionnalités implémentées

  • Inscription: Création de compte utilisateur
  • Connexion/Déconnexion: Gestion des sessions utilisateur
  • Gestion de profil: CRUD complet sur les profils utilisateurs
  • Sécurité: Politiques RLS pour protéger les données

Structure de la base de données

La table profiles a été créée avec les champs suivants :

Champ Type Description
id UUID Clé primaire, liée à l'utilisateur d'authentification
username text Nom d'utilisateur unique
full_name text Nom complet de l'utilisateur
updated_at timestamp Date de dernière mise à jour
